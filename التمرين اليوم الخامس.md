## Create Theme parks management  software   ,

Park Class :
rideID , rideName  , rideType , tickets, price

Validation :
rideID : 

1. Cannot be null 
2. Length more than 2

rideName :

1. Cannot be null
2. Length more than 4

rideType :

1. Cannot be null
2. can only have these values (rollercoaster|thriller|water)

tickets :

1. Cannot be null
2. must be a number

price:

1. Cannot be null
2. must be a number






Endpoints :
Get all the rides
Add new ride
Update ride
Delete ride
